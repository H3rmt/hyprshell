# hyprshell-launcher-lib

This crate is the heart of the launcher in hyprshell.
It builds the static and dynamic result sets, scores user input, and decides which item should be shown or launched.
It also contains action handling, calculator/path integration, and the custom matching logic for things like argument-taking actions.

- Main repo: https://github.com/h3rmt/hyprshell
- Docs: https://docs.rs/hyprshell-launcher-lib
