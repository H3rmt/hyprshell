# hyprshell-config-edit-lib

This crate provides the GUI-side helpers for editing hyprshell configuration.
It backs the settings editor with data transformation and component logic for the different config sections.
The app uses it to present editable launcher, plugin, and runtime settings in a more structured way.

- Main repo: https://github.com/h3rmt/hyprshell
- Docs: https://docs.rs/hyprshell-config-edit-lib
